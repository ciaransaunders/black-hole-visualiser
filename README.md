# Black Hole Visualiser

A collection of interactive black hole simulations and visualisations, built as standalone HTML files. Each program renders a physically-inspired black hole using Canvas 2D — no build tools, no dependencies, just open in a browser.

## Programs

### Event Horizon Series

| File | Description |
|------|-------------|
| `Event horizon Alpha.html` | Relativistic black hole visualiser (Gemini) |
| `Event horizon Beta.html` | Kerr black hole simulator v2 |
| `event_horizon.html` | Kerr metric simulator with controls for mass, spin, luminosity, disk height, temperature, and polar jets |
| `event_horizon_v3.html` | Upgraded Kerr simulator with tabbed panel UI and HUD physics readout |
| `event_horizon_v4.html` | Most advanced version — Doppler beaming, photon ring, EHT-style polarimetry, and preset support |

### Gargantua Series

| File | Description |
|------|-------------|
| `gargantua_v7_1.html` | Interactive Gargantua-style (Interstellar-inspired) black hole with parameter controls |
| `gargantua_oled_v2.html` | Gargantua visualisation optimised for OLED displays (pure black background) |

### Standalone

| File | Description |
|------|-------------|
| `singularity.html` | Kerr black hole simulation with blur-glass panel UI |
| `darkroom.html` | Artistic long-exposure photograph of a black hole, with crop marks and gallery framing |

## Usage

Open any `.html` file directly in a modern browser. No server or install required. Each visualisation runs at full viewport with floating control panels for adjusting parameters in real time.
